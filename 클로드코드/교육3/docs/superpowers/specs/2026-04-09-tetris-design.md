# 테트리스 게임 설계 문서

**날짜:** 2026-04-09  
**플랫폼:** 웹 (단일 HTML 파일)  
**기술 스택:** HTML + CSS + JavaScript (Canvas API)

---

## 개요

브라우저에서 실행되는 클래식 테트리스 게임. 단일 `index.html` 파일로 배포되며, 데스크탑 키보드 조작을 지원한다. 미니멀 클린 비주얼 스타일을 적용하고, 점수/레벨/최고점/홀드/넥스트 블록 등 풀 피처를 제공한다.

---

## 파일 구조

```
index.html  ← 모든 HTML, CSS, JS 포함 (단일 파일)
```

---

## HTML 레이아웃

```
┌─────────────────────────────────┐
│  HOLD        TETRIS   NEXT      │
│  [캔버스]    [캔버스]  [캔버스]  │
│  홀드블록    게임보드   넥스트   │
│              10×20    SCORE     │
│                       12,400   │
│                       LEVEL 3  │
│                       BEST     │
│                       45,200   │
└─────────────────────────────────┘
```

- 게임 보드 캔버스: 300×600px (셀 30px, 10열 × 20행)
- 홀드/넥스트 캔버스: 각 120×120px

---

## JS 모듈 구성 (단일 파일 내 6개 객체)

| 모듈 | 역할 |
|------|------|
| `PIECES` | 7가지 테트로미노 형태 배열 + 색상 정의 |
| `Board` | 10×20 그리드 상태, 충돌 감지, 줄 클리어 |
| `Piece` | 현재/다음/홀드 블록의 위치·회전 상태 |
| `Game` | 게임 루프(requestAnimationFrame), 점수·레벨 관리, 상태 머신 |
| `Renderer` | Canvas 그리기 (보드, 블록, 고스트 피스, UI) |
| `Input` | 키보드 이벤트 → Game 메서드 연결 |

---

## 게임 루프

```
requestAnimationFrame 콜백
  → 경과 시간 누적
  → 드롭 타이밍 도달 시:
      현재 피스 아래로 이동
      → 충돌 감지
        → 충돌: 보드에 고정 → 줄 클리어 → 새 피스 스폰
        → 미충돌: 계속 하강
  → 렌더링
```

---

## 점수 시스템

| 동시 클리어 줄 수 | 점수 |
|-----------------|------|
| 1줄 | 100 × 레벨 |
| 2줄 | 300 × 레벨 |
| 3줄 | 500 × 레벨 |
| 4줄 (테트리스) | 800 × 레벨 |
| 소프트 드롭 | 1점/칸 |
| 하드 드롭 | 2점/칸 |

---

## 레벨 & 속도

- 10줄 클리어마다 레벨 +1
- 드롭 간격: `max(100, 1000 - (레벨 - 1) × 80)` ms
- 레벨 1: 1000ms, 레벨 5: 680ms, 레벨 11+: 100ms (최대 속도)

---

## 홀드 규칙

- `C` 키로 현재 피스를 홀드 슬롯에 보관
- 홀드 슬롯에 피스가 있으면 교체
- 피스를 보드에 고정하기 전까지 한 번만 사용 가능 (연속 홀드 방지)

---

## 키보드 조작

| 키 | 동작 |
|----|------|
| ← / → | 좌우 이동 |
| ↑ / Z | 시계방향 회전 |
| ↓ | 소프트 드롭 (1칸, 점수 +1) |
| Space | 하드 드롭 (바닥까지, 점수 +2/칸) |
| C | 홀드 |
| P | 일시정지 / 재개 |

---

## 비주얼 스타일 (미니멀 클린)

**배경 & 그리드**
- 페이지 배경: `#FAFAFA`
- 보드 그리드 선: `#E0E0E0`
- 빈 셀: 투명 (배경색 그대로)

**테트로미노 색상 (머티리얼 팔레트)**

| 피스 | 색상 |
|------|------|
| I | `#29B6F6` (하늘색) |
| O | `#FFCA28` (노랑) |
| T | `#AB47BC` (보라) |
| S | `#66BB6A` (초록) |
| Z | `#EF5350` (빨강) |
| J | `#1565C0` (남색) |
| L | `#FF7043` (주황) |

**블록 렌더링**
- 모서리 `2px` 라운드
- 블록 상단/좌측에 밝은 하이라이트 (입체감)
- **고스트 피스**: 하드 드롭 위치를 동일 색상 20% 투명도로 표시

**게임 오버 오버레이**
- 반투명 흰색 오버레이 (`rgba(255,255,255,0.85)`)
- "GAME OVER" 텍스트, 최종 점수, "다시 시작" 버튼

---

## 데이터 저장

- `localStorage` 키: `tetris_best_score`
- 게임 오버 시 현재 점수 > 저장된 최고점이면 갱신

---

## 회전 알고리즘

기본 행렬 회전 사용 (SRS 월킥 없음). 회전 시 벽/블록과 충돌하면 회전 취소.

---

## 에러 처리

- 스폰 위치에서 즉시 충돌 발생 → 게임 오버 처리
- `localStorage` 미지원 환경 → 최고점 기능 비활성화 (try/catch)
