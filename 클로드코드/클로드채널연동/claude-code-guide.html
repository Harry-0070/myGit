설치 & 설정

Claude Code는 Node.js 기반 CLI 도구입니다. npm을 통해 전역 설치합니다.

1. 설치

bash 
# npm으로 전역 설치
npm install -g @anthropic-ai/claude-code

# 설치 확인
claude --version

2. API 키 설정

bash 
# 환경변수로 설정 (권장)
export ANTHROPIC_API_KEY="sk-ant-..."

# 또는 쉘 설정 파일에 영구 저장
echo 'export ANTHROPIC_API_KEY=sk-ant-...' >> ~/.bashrc
source ~/.bashrc

3. 프로젝트 폴더에서 시작

bash 
cd /your/project
claude
기본 사용법

기본 사용법

Claude Code는 자연어로 대화하듯 명령하면 됩니다. 프로젝트 폴더에서 실행하면 파일을 자동으로 인식합니다.

💬

자연어 명령

"이 함수를 최적화해줘", "버그 찾아줘" 등 대화체로 요청

📂

파일 자동 인식

현재 폴더의 코드를 자동으로 읽고 컨텍스트 파악

🔧

직접 수정

허락하면 파일을 직접 편집하고 명령 실행

🔄

대화형 세션

맥락을 기억하며 연속 대화로 작업 진행

기본 실행 모드

bash 
# 대화형 모드 (가장 일반적)
claude

# 단일 명령 실행 후 종료
claude -p "이 코드의 버그를 찾아줘"

# 특정 파일과 함께 시작
claude app.js

# 자동 승인 모드 (위험 주의)
claude --dangerously-skip-permissions
명령어 레퍼런스

슬래시 명령어

대화 중 /로 시작하는 명령어로 특수 기능을 실행할 수 있습니다.

명령어설명
/help도움말 보기
/clear대화 기록 초기화
/compact대화 요약해서 컨텍스트 압축
/cost현재 세션 토큰/비용 확인
/model사용할 Claude 모델 변경
/review코드 리뷰 요청
/commit변경사항 Git 커밋
/prPull Request 생성
/bug버그 리포트 작성
/initCLAUDE.md 프로젝트 설정 파일 생성
/memory기억 관리 (저장/조회)
/permissions도구 권한 설정 확인
/exit 또는 /quitClaude Code 종료
파일 & 코드

파일 작업

자주 쓰는 요청 패턴

대화 예시 
# 특정 파일 분석
> src/api.js 파일에서 에러 처리 개선해줘

# 새 파일 생성
> utils 폴더에 날짜 포맷팅 헬퍼 함수 파일 만들어줘

# 여러 파일 비교
> v1.js 와 v2.js 의 차이점 설명해줘

# 테스트 작성
> UserService 클래스에 대한 유닛 테스트 작성해줘

# 리팩토링
> 이 코드를 TypeScript로 변환해줘
💡

CLAUDE.md 활용: 프로젝트 루트에 CLAUDE.md 파일을 만들면 Claude가 항상 읽는 프로젝트 규칙/컨텍스트를 정의할 수 있습니다. /init 명령으로 자동 생성하세요.

팁 & 트릭

효과적인 프롬프트 작성

🎯

구체적으로 요청

"코드 고쳐줘" 보다 "login() 함수의 null 체크 추가해줘"처럼 구체적으로

📋

맥락 제공

에러 메시지, 스택 트레이스를 함께 붙여넣으면 더 정확한 답변

🔍

단계별 접근

큰 기능을 작게 나눠서 순차적으로 요청하면 품질 향상

✅

결과 검증

코드 수정 후 "테스트 실행해줘"로 바로 검증 요청 가능

파이프라인 활용 고급

bash 
# 파일 내용을 파이프로 전달
cat error.log | claude -p "이 에러 분석해줘"

# Git diff를 분석
git diff | claude -p "이 변경사항 코드리뷰 해줘"

# 출력 결과를 파일로 저장
claude -p "README.md 작성해줘" > README.md
⚠

주의: --dangerously-skip-permissions 옵션은 모든 파일 수정과 명령 실행을 자동 승인합니다. 신뢰할 수 있는 코드베이스에서만 사용하세요.
